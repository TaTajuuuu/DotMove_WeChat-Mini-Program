页面定义

加入页承载的核心任务

  • 展示待加入小组名称、生命周期和成员数。
  • 提示目标、打卡照片、备注和统计的同组可见性。
  • 要求用户设置小组内昵称。
  • 完成加入后进入目标设置流程。

关键规则映射

  • 仅未开始 / 进行中小组允许加入。
  • 已归档 / 已解散小组应阻止加入并给出明确提示。
  • 同一用户在同一小组中不得存在多个有效成员关系。
  • 已退出成员可重新加入；已被移除成员当前版本不可重新加入。

操作路径

  • 用户通过微信群分享卡片或邀请码进入本页。
  • 用户核对小组信息并填写昵称。
  • 确认加入后创建有效成员关系。
  • 加入成功后进入目标设置页，完成月度目标配置。
