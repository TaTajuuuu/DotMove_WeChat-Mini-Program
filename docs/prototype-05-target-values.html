页面定义

目标值页承载的核心任务

  • 直接填写上一页选择类型对应的目标值。
  • 未选择的目标类型不会在本页生成对应字段。
  • 运动天数和运动次数需要分别设置单天或单次最低热量。

关键规则映射

  • 目标配置按成员、小组、自然月维度整体保存。
  • 目标值为空、非数字或小于等于 0 时阻止保存。
  • 进行中小组首次保存后，整组目标配置锁定。
