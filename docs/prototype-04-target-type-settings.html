页面定义

目标类型页承载的核心任务

  • 选择一个或多个当前版本支持的目标类型。
  • 设置一点币承诺值；一点币仅展示，不参与结算、奖惩或统计。
  • 确认后进入目标值页面，仅为选择类型填写数值。

关键规则映射

  • 未开始小组可修改整组目标配置。
  • 进行中小组保存后不可新增、修改或删除目标配置。
  • 同一成员同一小组同一自然月内，同一目标类型只能设置一次。
